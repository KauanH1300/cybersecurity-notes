# O que é uma rede?

Uma rede de computadores é um conjunto de dispositivos conectados entre si com o objetivo de compartilhar dados, recursos e serviços.

Esses dispositivos podem se comunicar através de cabos, sinais sem fio ou fibra óptica.

---

## Exemplos de dispositivos em uma rede

- Computadores
- Celulares
- Servidores
- Impressoras
- Switches
- Roteadores

---

## Exemplos do dia a dia

- Acessar um site na internet
- Enviar mensagens no WhatsApp
- Assistir vídeos no YouTube
- Jogar online
- Compartilhar arquivos entre dispositivos

Tudo isso acontece através de redes de computadores.

---

## Objetivos de uma rede

As redes existem para permitir:

- Comunicação entre dispositivos
- Compartilhamento de arquivos
- Compartilhamento de internet
- Acesso a serviços
- Gerenciamento centralizado

---

## Como os dispositivos se comunicam?

Os dispositivos trocam informações através de protocolos de rede, que funcionam como regras de comunicação.

Essas informações são divididas em pequenas partes chamadas pacotes.

---

## Exemplo prático

Quando um usuário acessa um site:

1. O computador envia uma requisição pela rede
2. O roteador encaminha os dados
3. O servidor do site recebe a requisição
4. O servidor responde enviando os dados da página
5. O navegador exibe o conteúdo para o usuário

---

## Resumo

Uma rede permite que dispositivos se conectem e troquem informações de forma organizada e eficiente.
